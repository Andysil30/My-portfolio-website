# Andrew Mwiti — Portfolio Website

A product-focused portfolio website showcasing web development and ML engineering work.

## Latest Deployment
- **Commit:** [592a232a](https://github.com/Andysil30/My-portfolio-website/commit/592a232ac0258ed45a20692da080389f663bfb78)
- **Date:** 2026-02-25
- **Changes:** Initial portfolio website setup with hero section, about, portfolio projects, blog, and contact form

## Features
- Responsive design (mobile-first)
- Dark theme with glassmorphism
- Project filtering and search
- Modal project details
- Contact form with Formspree integration
- Smooth scroll navigation
- Back-to-top button
- Accessibility-first approach (ARIA labels, semantic HTML)
- Performance optimized

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** CSS Grid, Flexbox, CSS Variables
- **Forms:** Formspree integration
- **Images:** WebP with fallback support

## Getting Started

1. Clone the repository
2. Open `website.html` in your browser
3. Customize with your own content, images, and links

## Customization
- Update social links in header and footer (GitHub, LinkedIn)
- Replace project images in the `images/` folder
- Update contact form action URL (Formspree)
- Modify color scheme in CSS variables (`:root`)
